Python Implementation of Red Eye Correction and Image enhancement functions (grayscale) with a user friendly GUI.

Run the python file through the command - python gui.py 

Instruction to operate GUI:

Three buttons can be seen initially.
Use the 'Select Image' button to browse and choose an image from the local directory.
The image gets loaded on the GUI.
Now, Click the button corresponding to a function to be implemented on the selected image.

Repeat the steps above to test on another image. Close the window to exit.

Caution:
An image must be selected and loaded on the GUI before any functions can be implemented.
Red Eye correct function must be operated only on color images. 
For Auto Enhancement, the output is always grayscale.
